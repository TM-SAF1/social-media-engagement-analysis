# YouTube Trending Video Analysis

## Dataset
- Trending YouTube Video Statistics (US) from Kaggle
- CSV file + JSON category mapping

## Tools
- Python
- Pandas
- Streamlit
- Matplotlib

## Methodology
1. Loaded CSV and JSON data
2. Mapped category IDs to names
3. Cleaned data & fixed dates
4. Created engagement metric (like_ratio)
5. Built charts to visualize category trends, engagement, and time-based patterns
6. Extracted key insights

## Key Findings
- A few categories dominate trending videos (Music, Entertainment, Gaming, etc.)
- Views follow a heavy-tail distribution: most videos low views, few go viral
- Likes strongly correlate with views → engagement matters
- Trending patterns fluctuate over time → user interest changes
